# BusinessDanmaku_Category

## 简介

BusinessDanmaku_Category 是一个私有 CocoaPods 库。

## 安装

在你的 `Podfile` 中添加：

```ruby
pod 'BusinessDanmaku_Category', :git => 'https://github.com/iOSComplexBusiness/BusinessDanmaku_Category.git'
```

然后运行：

```bash
pod install
```

## 使用

```objc
// 导入头文件
#import "BusinessDanmaku_Category.h"
```
